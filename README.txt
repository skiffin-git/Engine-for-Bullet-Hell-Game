TO START THE GAME:

1) Download project generator from my drive: https://drive.google.com/file/d/1rXljIMNWSPay4YjwDIE_894n2kS3KtF5/view?usp=sharing
2) Generate new project with openframework and choose ofxGUI addon 
2) Replace all file in src folder
3) (for IntelIJ) Open project in IDE and rightclick on src folder, choose add existing files
4) Go to src files, select all files from src and click add
5) Replace all file in bin/data folder
6) Open .sln file and start a debug
7) Play
